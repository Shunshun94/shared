ポジションマッピング 編集画面

説明
出力
マップを追加

いわゆる「レーダーチャート」とか「クロスチャート」とか言われる十字上のマップにキャラ駒を配置してキャラクターの紹介をするものを生成するアプリです。業界用語では「ポジションマッピング」と呼ぶそうです。

既にマップが作られている場合はここから編集が可能です。

また、「マップを追加」をクリックすることで新しいマップを追加できます。

編集したポジションマッピングの保存はゆとシート上で行います。「出力」のタブを開き、そこに書かれたテキストをキャラクターシートの「容姿・経歴・その他メモ」に貼り付けてください。「容姿・経歴・その他メモ」に情報が含まれていれば十分なので折りたたむなどして普段は隠しておいても構いません。

完成したものを閲覧する際はこちらからどうぞ。

ソースコードはこちら。

以下をコピーして、キャラクターシートの「容姿・経歴・その他メモ」欄にそのまま編集せずに貼り付けてください。「容姿・経歴・その他メモ」欄に他の記述が含まれていても構いません。
