# オンセ向け威力表運命変転ツール

SW2.X のオンセでダメージ計算時に運命変転を使った際にどんなコマンドで再計算するのか教えてくれるツール

## 使い方

[こちら](https://shunshun94.github.io/shared/other/io/github/shunshun94/trpg/sw2/FateAlteration/index.html)にアクセスして利用。

## 他のツールに統合する

[こちら](./fateAlteration.js)を読み込んで `io.github.shunshun94.trpg.sw2.FateAlteration.calc` を実行。
