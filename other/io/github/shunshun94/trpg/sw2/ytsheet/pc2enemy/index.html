ソードワールド2.5 PCをエネミーデータに変換

使い方
  1. ゆとシートの URL をキャラクターシート URL に入力して「実行」をクリック
  2. ダウンロードされたファイルをゆとシートのコンバートの「ローカルファイルから」でアップロード・作成
すでにわかっている課題

将来的に直るかもしれませんし直らないかもしれません。上の方に記載されているものの方が今後修正される可能性が高いかもしれませんしそんなことはないかもしれません。

  • バルバロスレイジ以降のサプリに全く対応していない
  • シールドバッシュと投げに命中強化等の補正が乗らない。技能レベルと器用度ボーナスのみで計算している
  • ステータスに反映されていない装飾品の効果が反映されない
  • その他の持ち物（魔晶石等）の効果が反映されない
