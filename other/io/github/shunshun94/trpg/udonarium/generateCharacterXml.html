ゆとシート2からユドナリウムの駒を作るツール for SW2.5

URL:

　

使い方

  1. ゆとシート2で作ったキャラクターシートまたは魔物シートの URL を URL 欄に入れる
  2. 『生成』と書かれたボタンを押す
  3. ダウンロード data_(キャラクター名).xml と書かれたリンクが出てくるのでクリックして保存する
  4. 保存されたファイルをユドナリウムにドラッグ & ドロップする
